# How To Contribute

Contributions to these docs, or any of our open source projects is warmly welcomed!

You can find others developing on Streamr, as well as get in touch with the Streamr core contributors: <a target="_blank" rel="noopener noreferrer" href={routes.community.discord()}>Discord</a>

You can also reach out to the core team over [email](mailto:contact@streamr.com)! Here is a <a href="https://github.com/streamr-dev/core-frontend/blob/master/app/src/docs/docsEditingGuide.md" target="_blank" rel="noopener noreferrer">guide on contributing to the Docs</a>.
